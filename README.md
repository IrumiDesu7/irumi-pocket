# Irumi Pocket

Aplikasi belajar Islam berbasis web (PWA) dengan manhaj Salafi. Menggabungkan tiga sumber materi — Panduan Adab Sunnah, Bacaan Shalat, dan Panduan Tajweed — dalam satu aplikasi yang bisa diakses offline.

## Fitur

- Panduan Adab Sunnah berdasarkan hadits shahih
- Bacaan Shalat lengkap dengan teks Arab, transliterasi, dan terjemahan
- Panduan Tajweed dengan contoh dan penjelasan
- Bookmark / favorit untuk menyimpan konten penting
- Bisa diinstall sebagai aplikasi (PWA) dan diakses offline
- Tampilan responsif untuk mobile dan desktop

## Tech Stack

React 19 · Vite · Tailwind CSS 4 · TypeScript · Workbox PWA

## Development

```bash
pnpm install
pnpm dev
```
